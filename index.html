Blackberry.js

A zero-build framework for the artisanal web.

Overview

Blackberry is a tiny frontend library that gives you reactivity and composability. Scoped styles, optional JSX rendering, and a simple API make it a fun choice for building websites and web apps.

Blackberry.js is inspired by Strawberry.js, and is designed to be used as a lightweight html-first alternative to frameworks like Vue and Svelte without requiring build tooling or a complex dependency tree, while still providing a component model, scoped styles, and reactivity.

Setup

The ideal way of using Blackberry.js is by copying the source code into your project directory.
